Quill

Quill component is a copy of ngx-quill component with small fixes that let prevent app crash in IE10. So you can find the documentation here:
https://github.com/KillerCodeMonkey/ngx-quill
Don't forget to sanitize HTML output on the server side to avoid XSS vulnerability.
Quill uses .ql-* classes for formatting. To display content properly you need to include vendor/libs/quill/typography.scss file and place Quill-generated content within .ql-content element.
